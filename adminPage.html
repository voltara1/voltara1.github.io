Admin Dashboard

Admin dashboard

Select an action on the left to manage users or tutorials.

Users management overview

List ALL users will show a table with:

  • User id, username, email address
  • Creation or registration date
  • Number of tutorials submitted by this user
  • Total number of likes received

Sorting controls will allow ordering by user id, username, email, registration date and total likes.

Tutorial management overview

options include:

  • List tutorials by category with checkboxes for Arduino, ESP32, IoT, PCB Design, Raspberry Pi, Robotics, Sensors and STM32 and controls to select or clear all categories.
  • List of curated tutorials with sorting by number of likes in ascending or descending order.
  • List of non-curated tutorials with sorting by number of likes in ascending or descending order.
  • List of tutorials by proficiency with checkboxes for Beginner, Intermediate and Advance and support for multiple selections.

Future JavaScript will render the tables and filters in this area based on the selected menu item.
